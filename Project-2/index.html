Emotion Detection Based on Text and Emojis

An innovative emotion detection model that combines text and emoji data, enhancing accuracy in emotion recognition. This adaptable model finds applications in sentiment analysis, feedback assessment, mental health monitoring, and user-centric contexts, offering valuable insights for improved digital communication and meeting evolving user and organizational needs.



Team members include: Anushka, Shanaya, Nicole

Features

GloVe Word Embedding:

Utilizes GloVe (Global Vectors for Word Representation) to transform textual data into continuous vector space, capturing semantic relationships among words based on global word-word co-occurrence statistics.

Long Short-Term Memory (LSTM):

Implements LSTM, a specialized recurrent neural network (RNN), to capture sequential information in texts effectively, overcoming the gradient vanishing problem through gating mechanisms, enhancing the model's ability to understand and analyze textual content for sentiment and emotion detection.

Text Preprocessing:

The model preprocesses input text by tokenizing, removing punctuation, and converting to lowercase to ensure accurate analysis.
